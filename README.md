# 🛡️ Insurance Data Analysis 

## 🔍 Overview

This is a **personal end-to-end data analysis project** focused on the Insurance dataset.
The project demonstrates how raw data can be transformed into **meaningful business insights** using multiple tools and technologies.

It covers the complete analytics pipeline — from **data cleaning to visualization and insights generation**.

---

## 🎯 Objectives

* Analyze **insurance charges and claim patterns**
* Study impact of **age, BMI, smoking habits, and region**
* Identify **high-risk customer segments**
* Perform **data querying and transformation**
* Build **interactive dashboards** for insights

---

## 🧩 Project Modules

### 1️⃣ Excel (Data Cleaning & Preprocessing)

* Handled missing values and duplicates
* Structured raw dataset
* Created basic summaries using pivot tables

---

### 2️⃣ SQL (Data Analysis & Querying)

* Wrote queries to extract insights
* Used aggregations (SUM, AVG, COUNT)
* Performed grouping and filtering
* Generated business-related metrics

---

### 3️⃣ Power BI (Dashboard & Visualization)

* Built interactive dashboards
* Created KPIs for:

  * Total Charges
  * Average BMI
  * Customer distribution
* Used filters and slicers for dynamic analysis

---

### 4️⃣ Tableau (Advanced Visualization)

* Designed interactive dashboard (`.twbx`)
* Visualized:

  * Charges vs Age
  * BMI impact
  * Smoker vs Non-smoker comparison
* Applied storytelling techniques

---

## 🛠️ Tools & Technologies

* **Excel** → Data Cleaning & Preparation
* **SQL** → Data Querying & Analysis
* **Power BI** → Dashboard & KPI Visualization
* **Tableau** → Advanced Visual Analytics

---

## 📊 Key Insights

* 💰 Smokers incur **significantly higher insurance costs**
* 📈 Charges increase with **age and BMI**
* ⚠️ High-risk group: **Smokers with high BMI**
* 🌍 Regional variation observed in charges
* 👥 Gender has **minimal impact on charges**
