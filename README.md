# 🛡️ Insurance Data Analysis Project

## 🔍 Overview

This project analyzes **insurance data** to uncover insights related to customer demographics, policy trends, and claim behavior.

The goal is to transform raw data into **interactive visual insights** using Tableau for better business decision-making.

---

## 🎯 Objectives

* Analyze **insurance charges and claim patterns**
* Understand impact of **age, BMI, smoking, and region**
* Identify **high-risk customer segments**
* Visualize **cost distribution and trends**
* Build an **interactive Tableau dashboard**

---

## 🛠️ Tools & Technologies

* **Tableau** → Dashboard creation & visualization
* **Excel / CSV** → Data source
* **Data Analysis Concepts** → Data cleaning, aggregation, filtering
--

## 📊 Key Insights

* 💰 **Smokers have significantly higher insurance charges**
* 📈 Charges increase with **age and BMI**
* 🌍 Regional differences observed in **insurance costs**
* ⚠️ High-risk group identified: **Smokers + High BMI**
* 👨‍👩‍👧 Gender impact is relatively **less significant**
